# 4X1-MUX-using-Microwind

Overview

This repository contains the design and simulation of a 4×1 Multiplexer (MUX) implemented using Microwind. The project demonstrates transistor-level CMOS implementation, layout design, and simulation of digital logic circuits in VLSI design.

A 4×1 Multiplexer selects one of four input signals and forwards it to a single output line based on the select inputs.

Objectives:

Design a transistor-level 4×1 MUX

Create CMOS layout using Microwind

Perform functional simulation

Analyze output waveforms

Understand VLSI layout concepts and CMOS implementation

Software Used

Microwind
CMOS Technology Library
